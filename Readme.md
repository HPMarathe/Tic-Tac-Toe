## 1D Arrays

```
let arr1 = ["apple", "orange", "Mango"];

```

How to access anything on 1d array.
arr1[0]

## 2D Array

```
let arr2 = [
["banana", "litchi", "Mango"],
["bottle", "Vegetable"],
["watch", "Notch"],
];
```

How to access anything on 2d array.

arr2[0] - ["banana", "litchi", "Mango"]

arr2[0][0] - "banana"
